# ❄️ SubZero — Subscription Audit & Cancel Assistant

> Stop paying for subscriptions you forgot about.

SubZero helps you track every subscription you pay for, tag what to keep or cut, and cancel the ones draining your money — with step-by-step guides. Free. No account. No server. Everything stays in your browser.

## ✨ Features

- 📊 **Live spending dashboard** — See your monthly and yearly totals update in real-time
- 🏷️ **Tag subscriptions** — Mark each one as Keep, Review, or Cancel
- ✏️ **Edit & delete** — Update costs when prices rise, remove what you've cancelled
- 🔍 **Search, filter & sort** — Find anything instantly by name, status, or cost
- 📖 **Cancel guides** — Step-by-step cancellation instructions for Netflix, Spotify, Adobe, Amazon and more
- 💱 **Multi-currency** — Switch between EUR, USD, GBP, TRY, CHF, PLN
- ⬇️ **Export to CSV** — Download your full list for Excel or Google Sheets
- 🌙 **Dark & light mode** — Toggle between themes, preference saved automatically
- 🔒 **100% private** — All data stays in your browser via localStorage. No account, no server, no tracking

## 🚀 Getting Started

### Run locally

No build tools. No npm. No framework. Just open it.

```bash
git clone https://github.com/sraboni-bhuiyan/subzero.git
cd subzero
```

Then open `index.html` in your browser — or use VS Code's **Live Server** extension for hot reload.

### Deploy to Netlify

1. Push your code to GitHub
2. Go to [netlify.com](https://netlify.com) → **Add new site** → **Import from Git**
3. Select your repository
4. Leave build command and publish directory **empty**
5. Click **Deploy**

Every `git push` to `main` redeploys automatically.

## 📋 How It Works

1. **Add** your subscriptions manually — name, cost, category, billing cycle
2. **Tag** each one as Keep, Review, or Cancel
3. **See** your potential savings update instantly on the dashboard
4. **Cancel** with a step-by-step guide per service
5. **Export** your full list as CSV anytime
